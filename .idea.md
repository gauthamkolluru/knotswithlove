# Knots With Love (knotswithlove.org) a web application for predominantly selling crochet patterns and products (in tiny scale) at times.

## what do we want from this project

- a well sustainable and easily maintainable website requiring minimal software changes but could sustain for a maximal years of time.
- a modular website that can be extended and modified in the future for the future.
- a highly secure website that can fight hacker attempts and is resilient.
- the website must be efficient in handling multiple users in parallel and multiple requests effectively and efficiently in parallel with efficient implementation of asynchronous and multi-processing.
- evaluate the current architecture on all important aspects like software principles, overall technical architecture, security, resilience, asynchronicity and multi-processing capailities and everytime a significant change is made, evalutate the modified system again, all on the scale of 1-100 and anytime the score is less 85, rework on that aspect of the module to improve its score until it goes equal to or beyond 85.
- always document the vulnerabilities of the code and fix them in the code and if there are any vulnerabilities in the code that cannot be fixed by coding, document and highlight them and provide some suggestions to how to go about them.
- we will soon be implementing payment gateway for this web application, ensure all safety and security measures are maintained even for them and in all probability, we might end up using a simple third party code implementing the same and the making that part of the secure even then is also to be ensured.
- test the code throughly to work for 1000 users in parallel at the least all trying to add checkout products from the website without issues.
